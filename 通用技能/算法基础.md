# 算法基础知识 

* ## 时间复杂度

> 衡量算法耗时的指标 **大O符号表示法** : `T(n) = O(f(n))`
>
> 其中`f(n)`表示每行代码执行次数之和， 而 O 表示正比例关系
>
> 大O符号表示法并不是用于来真实代表算法的执行时间的，它是用来表示代码执行时间的增长变化趋势的。

例如
```
for(i=1; i<=n; ++i) {
   j = i;
   j++;
}
时间复杂度为 ： O(n)
```

* ## 空间复杂度

> 对一个算法在运行过程中临时占用存储空间大小的一个量度，
>
> 同样反映的是一个趋势，我们用 S(n) 来定义。

例如
```
int[] m = new int[n]
for(i=1; i<=n; ++i) {
   j = i;
   j++;
}
空间复杂度为： S(n)
```
